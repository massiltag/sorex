### How to install ###
* Compiling sources
```bash
javac -cp ./ util/*.java
javac -cp ./ blockchain/*
javac -cp ./ architecture/*
```
* Launching Client
```bash
java architecture/SorexClient
```
* Launching Server
```bash
java architecture/SorexClient
```
